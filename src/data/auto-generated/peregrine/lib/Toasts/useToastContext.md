A hook that provides access to the toast state and dispatch.
Any component using this hook _must_ be a child of a [ToastContextProvider](#ToastContextProvider).

**Returns:**
`Array.<Object>`

**Example**  

```js
const [toastState, dispatch] = useToastState();
```
